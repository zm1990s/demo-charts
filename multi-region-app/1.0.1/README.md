# 支持多云环境部署的 demo app

允许用户选择当前的应用应该部署在哪个云，根据不同云使用不同的 image 地址、configmap、service、PVC等 来创建应用。



# 配置调研表模板

| 项目                                            | AKS                   | CCE                   | ACE               | 对应Charts表单设计                           |
| ----------------------------------------------- | --------------------- | --------------------- | ----------------- | -------------------------------------------- |

| Pod数量                                         | 2                     | 3                     | 不在此云部署      | 在表单中暴露，并指定默认值                   |

| PVC/PV                                          | 使用已有的，名称为xxx | 使用已有的，名称为xxx | 不在此云部署      | 不进行配置，只调用已有的                     |
